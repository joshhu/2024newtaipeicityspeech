# 20年內不被AI取代的求生技能

## 內容簡介
隨著人工智慧的發展，許多傳統的工作正在面臨被自動化和智能化取代的風險。然而，仍然存在一些關鍵的求生技能，即使在 AI 普及的未來，依然無法被取代。我們將發現那些不可替代的技能，並學習如何培養和應用它們，以因應快速變遷的數位時代。

## 課程目標
- 開放問答式的課程，讓學員能夠針對自己的需求提問
- 了解人工智慧的基本概念
